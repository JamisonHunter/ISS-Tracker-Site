About issdash.net

This application tracks the current location of the International Space Station. The API updates every 5 seconds allowing for nearly real time location data! Given the remarkably fast orbital velocity of the ISS, it is easy to see the space station move from place to place within minutes. This web application was built using a C# backend and a simple JavaScript based frontend. For those interested, the source code can be found below. Thanks for stopping by!
